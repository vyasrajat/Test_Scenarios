# 🧪 Object-Based Test Scenario Project

## 📌 Project Overview
This project demonstrates **test scenario writing skills** using real-world objects and applications.

Each object is analyzed with:
- Positive Scenarios ✅
- Negative Scenarios ❌

The goal is to improve **QA thinking, edge case identification, and real-world testing approach**.

---

## 🎯 Objective
- To practice writing effective test scenarios  
- To understand product behavior in real-world conditions  
- To identify both expected and unexpected outcomes  

---

## 🧪 Scenario Types

### ✅ Positive Scenarios
- Verify expected behavior  
- Ensure smooth functionality  
- Validate correct output  

Example:
- Pen writes smoothly  
- Fan rotates properly  
- ATM dispenses cash correctly  

---

### ❌ Negative Scenarios
- Verify failure conditions  
- Check system behavior under invalid conditions  
- Identify possible defects  

Example:
- Pen leaks ink  
- Elevator stops working  
- Payment fails in WhatsApp  

---
